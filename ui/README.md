# UI Prototype

This folder contains a static HTML prototype for the CloudSigma Image Lifecycle UI.

## Files
- `index.html` - main prototype

## Open locally
You can open it directly in a browser, for example:

```bash
xdg-open /home/cloudsigma/cloudsigma-image-lifecycle/ui/index.html
```

Or serve it with a simple static server:

```bash
cd /home/cloudsigma/cloudsigma-image-lifecycle/ui
python3 -m http.server 8099
```

Then visit:

`http://localhost:8099`

## Purpose
This is a visual prototype only.
It is not connected to the backend yet.
